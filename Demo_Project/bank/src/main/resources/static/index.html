ABOUT THE APPLICATION

This application is built inorder to provide online banking support to users, by creation of a savings account. Users can send funds to other accounts who have registered in the application. The backend part of this applicatin has been built using SpringBoot Java REST API.

Also further extending the knowledge, we used NeonDB as the database(PostgreSQL). The application is finally supposed to be deployed in OpenStack.

Modules used: Register -> Login, Transaction, Account Handling, Payment

DB and deployment: Neon DB, OpenStack
